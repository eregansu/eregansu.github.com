# AuthError

Exception class whose instances are thrown when an authentication exception
occurs.

<code>AuthError</code> is a class derived from <a href="Exception">Exception</a>.

## Synopsis

<pre><code>uses('auth');

throw new AuthError($engine, $message, $reason);
